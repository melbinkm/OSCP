Locate JMP ESP
==============
View Modules
```
!mona modules
```
Find OPCode
```
msf-nasm_shell
nasm > jmp esp
00000000   FFE4         jmp esp
````
Find JMP ESP in Module
```
!mona find -s "\xff\xe4" -m "foundfile.dll"
````
Goto JMP ESP and set Breakpoint
```
Go to address in Disassembler > Enter Address Found > Set Breakpoint > Run script and check
```
